WinLLDPService

WinLLDPService is free, open source and tiny Windows service. It sends Windows' machine network information via LLDP so that network administrators can find computers with ease.

WinLLDPService acts only as an announcer to the switch (sends packets to the switch). It will not query switches' internal LLDP data.

Possible errors are logged to Windows Event Log.

Requirements:

  • WinPcap

Download

Download WinLLDPService-installer.msi. Source can be found at GitHub. Report possible bugs and feature requests to issues page.
